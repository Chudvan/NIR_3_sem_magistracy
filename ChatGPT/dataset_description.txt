6) Выдвинута гипотеза, что признаки из списка pa_fields получены с помощью метода МГК (PCA) из признаков seven_fields 
7) столбцы facs_fields содержат данные из 42-мерного пространства Facial Action Coding System (FACS) // action units
8) столбцы seven_fields содержат данные из 6-мерного пространства Пола Экмана (6 эмоций + Neutral)
9) столбцы pa_fields содержат данные из 2-мерного пространства Valence-Arousal (или Pleasure-Arousal). Это пространство является подпространством VAD (PAD) // координата Dominance опущена.
